## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Jivan2801/tic-tac-toe
cd tic-tac-toe
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

The app will run at **http://localhost:3000** (or another port if 3000 is busy).
